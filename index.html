Energy Consumption Prediction

Time Series Analysis of energy consumption in a low energy building.

This project uses an Appliances Energy Prediction Dataset from UCI Machine Learning Repository. The objective of the project was to predict the energy consumption of the appliances using Linear Regression and Support Vector Regression(SVR). There is a significant increase in energy consumption from the 16th hour and this peaks at the 18th hour. Monday is the day with the most energy consumption closely followed by Friday, Saturday and Sunday. Though both models had similar accuracy scores, the SVR model outperformed the Linear model with a significantly higher coefficient of determination regression score.

Social Media Sentiment Analysis

Using Machine Learning Algorithms For Sentiment Classification

In this project, the Twitter US Airline sentiment dataset obtained from Kaggle was utilized for sentiment analysis. The objective of the project was to compare the performance of four different Machine Learning Algorithms (Random Forest, Naïve Bayes, Logistic Regression, and Support Vector Machine) in performing sentiment classification based on different Data split ratios, Feature engineering, and Random Minority Oversampling technique. It was observed that, the 80:20 data split ratio, in addition to Feature Engineering improved the classification performance of the Algorithms. While the Random Minority Oversampling technique improved the recall score for the Positive sentiments for most Algorithms. The Support Vector Machine classifier attained the highest Accuracy of 93% and F1-score of 87%.

Breast Cancer Prediction

Breast Cancer Prediction Using Supervised Machine Learning.

The performance of five machine learning algorithms—Support Vector Machine (SVM), K-Nearest Neighbour (KNN), Random Forest (RF), Logistic Regression (LR), and Decision Tree (DT)— were evaluated in predicting Breast Cancer. The main goal was to determine the most effective algorithms based on performance metrics like Confusion Matrix, Accuracy, and Precision. The results showed that SVM achieved the highest accuracy of 98%, outperforming other classifiers. The experiments utilized the Breast Cancer Wisconsin Diagnostic Dataset from the UCI Machine Learning Repository. Employing a supervised machine learning approach, the models were trained on labeled class data distinguishing between Malignant and Benign tumors. This research aimed to enhance disease diagnosis accuracy, ultimately contributing to better treatment outcomes and patient survival.

Fuel Efficiency Prediction

Python, DT, RF, XGBoost

This project uses an Auto MPG dataset from Kaggle to build a model that can predict miles per gallon(MPG) which is the primary measurement of a car's efficiency. Three regression algorithms; Decision Tree, Random Forest and XGBoost were used to train the model.Scikit learn's built in mean_squared_error method was used to test the regression models and math.sqrt was used to get the root mean squared error(RMSE). XGBoost performed the best with an RMSE value of 2.3 while Decision Tree and Random Forest had an RMSE value of 2.4 and 3.3 respectively.
